# cooking-a-snake

A text game in Python where you have to cook a snake before it can poison you. 

### The rules ###

- After you are bitten 3 times, the game is over
- If the snake is cooked, you win

### How to cook the snake ###

- Light the fire
- Get the snake
- Put the snake in the fire

### Controls ###

Use **WASD** to move.
